¿Qué es una Función?

Un bloque de código reutilizable que realiza una tarea específica. Como una mini-receta.

2 / 15

Definir una Función: def

Se usa la palabra clave `def`, un nombre, paréntesis y dos puntos.

def saludar():
    print("¡Hola, mundo!")
3 / 15

Llamar a una Función

Para ejecutar la función, simplemente escribe su nombre seguido de paréntesis.

# Primero se define
def saludar():
    print("¡Hola, mundo!")

# Luego se llama
saludar()
4 / 15

Parámetros

Son variables que reciben valores cuando se llama a la función.

def saludar_a(nombre):
    print(f"¡Hola, {nombre}!")

saludar_a("Ana")
5 / 15

Argumentos

Son los valores que se envían a la función al llamarla (ej. "Ana").

# 'nombre' es el parámetro
def saludar_a(nombre):
    print(f"¡Hola, {nombre}!")

# "Ana" es el argumento
saludar_a("Ana")
6 / 15

Retornar un Valor: return

Una función puede devolver un resultado para ser usado en otra parte.

def sumar(a, b):
    return a + b

resultado = sumar(5, 3)
print(resultado) # 8
7 / 15

Argumentos por Defecto

Se puede asignar un valor por defecto a un parámetro.

def saludar(nombre="Mundo"):
    print(f"¡Hola, {nombre}!")

saludar() # Imprime "¡Hola, Mundo!"
saludar("Python") # Imprime "¡Hola, Python!"

Funciones que Python ya trae listas para usar, sin necesidad de definirlas.

Built-in: print()

Built-in: len()

Devuelve la longitud (número de elementos) de un objeto.

print(len("hola")) # 4
print(len([1, 2, 3])) # 3

Built-in: max(), min(), sum()

Operaciones matemáticas comunes sobre secuencias de números.
